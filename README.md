ACADEx

Academic Centralized Analytics & Data for Student Expense Management

Overview

ACADEx is a data-driven web application built using Python and Streamlit that enables students to manage and analyze their academic performance, attendance, study habits, and expenses in a single platform. The application provides meaningful insights through data visualization and basic predictive analysis to support better academic and financial decision-making.

Features

- Track academic performance (marks)
- Monitor attendance
- Analyze study hours
- Manage and track expenses
- Interactive data visualization
- Basic predictive analysis using machine learning
- SQLite database integration

Tech Stack

- Frontend: Streamlit
- Backend: Python
- Database: SQLite
- Libraries: Pandas, NumPy, Scikit-learn, Plotly

Installation and Setup

1. Clone the repository
   git clone https://github.com/ErKrishnaarora/ACADEx.git

2. Navigate to the project directory
   cd ACADEx

3. Install dependencies
   pip install -r requirements.txt

4. Run the application
   streamlit run app.py
